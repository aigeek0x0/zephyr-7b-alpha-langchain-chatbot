This is Gradio Chatbot that operates on Google Colab for free. You can utilize it to chat with PDF files saved in your Google Drive.

**Tech Stack**
Zephyr 7B Alpha (Finetuned Mistral 7B Instruct)
Langchain
HuggingFace
ChromaDB
Gradio
