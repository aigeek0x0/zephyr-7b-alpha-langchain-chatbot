This is Gradio Cahtbot that runs on Free Google Colab. You can use this to chat with PDF files stored on your Google Drive.

**Tech Stack**
Zephyr 7B Alpha (Finetuned Mistral 7B Instruct)
Langchain
HuggingFace
ChromaDB
Gradio
